    Learn Code

    Online Coding courses

    Codecademy is a smart site designed to help you interactively learn code.

    Microsoft Virtual Academy offers courses in IT, Cloud Computing, etc, for students of all levels. Register for live events like 2D game programming in HTML5.

    Stack Overflow is an open enviorment for people to ask questions about code problems, and people to answer for debugging purposes.

    Github is an open source that allow people to collaborate in a project, upload your own project and share it with other people.

    Atom is a free and open-source text and source code editor for macOS, Linux, and Microsoft Windows with support for plug-ins written in Node.js, and embedded Git Control, developed by GitHub.

    MIT App Inventor allows "Anyone Can Build Apps That Impact the World", App Inventor for Android is an open-source web application. Many resources is available on their website, which their MIT App Inventor 2 especially provides fundamental knowledge of code, block-building code strategy that allows users to create apps on androids devices. Moreover, there is also documentations, tech events and articles available which makes this web a awesome tool for students to get in touch with.

    w3schools "W3Schools is a web developers site, with tutorials and references on web development languages such as HTML, CSS, JavaScript, PHP, SQL, Python, W3.CSS, and Bootstrap, covering most aspects of web programming."

    Bootstrap"Build responsive, mobile-first projects on the web with the world's most popular front-end component library.Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery."

    Academics

    Improve your academic subjects online.

    Alison brings you courses in many areas that you can explore. It takes top, reputable materials and makes them super accessible.

    edX features university level courses from MIT, Dartmouth, Cornell, and more. Some courses cost money, but the majority are free to learn.

    MIT OpenCourseware is a more in depth selection of university level courses. They offer a wide range of courses, including animal behavior and statistics and probability.

    Coursera combines more than 100 courses from elite institutions. It holds a colossal library of courses from countries all over the world, such as Stanford, Duke, and the University of Edinburgh, etc.

    Video Courses

    Learn from videos.

    Khan Academy offers step by step videos on a particular subject area.

    CrashCourse is similar to Khan Academy, it provides tutorials for a specific subject

    Study Guides

    Online Study Guides to follow.

    Quizlet allows you to create unique study guides to accommodate your learning preferences.

    CliffsNotes supplements reading with brief summaries of chapters-hundreds of books available. Not to be used as an alternative to reading books.

    Shmoop, similar to Cliffnotes, supplements reading with brief summaries of chapters-hundreds of books available. Not to be used as an alternative to reading books.

    New Languages

    Explore a new way of communicating by learning the mother tongues of foreign countries

    Duolingo is a revolutionary and intuitive language-learning tool that provides step-by-step instructions on acquiring language skills, whether you are a beginner, intermediate, or advanced speaker.

    Memrise is a user-generated learning platform which uses flashcards as memory aids. It is known for the language learning service, but it also offers content on a wide range of other subjects. Memrise has more than 150 language courses available to learn.

    Organize

    Tips for better time management.

    Time Management For The Time-Challenged is a Forbes article on how to better manage your time.

    How To Use A Planner (Without Getting Totally Overwhelmed) is a HuffingtonPost article on how to plan your time.
